# Nine Dragon Sports Development Company — Multi-language Website (EN / 中文 / 日本語)

This is a **static** website package designed for **GitHub Pages**.

## 1) Publish on GitHub Pages (recommended workflow)

1. Create a new GitHub repository (e.g., `ninedragon-website`).
2. Upload **all files and folders** from this ZIP to the repo root.
3. In GitHub: **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`
4. Save. GitHub will generate a Pages URL.

## 2) Language switching

- Use the top-right buttons: **EN / 中文 / 日本語**.
- The site stores the selected language in `localStorage`.
- Translation files are in `/i18n/en.json`, `/i18n/zh.json`, `/i18n/ja.json`.
